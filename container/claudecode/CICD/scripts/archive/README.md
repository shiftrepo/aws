# Archive - 使用禁止スクリプト

このディレクトリには、環境を破壊する可能性のある過去のスクリプトが保管されています。

## ⚠️ 使用禁止

### trigger-cicd-mr.sh

**使用禁止理由:**
- 既存のブランチとコミット履歴を上書きする
- localhost撲滅などの重要な修正を破壊する
- 新しいブランチを勝手に作成してしまう

**問題が発生したケース:**
- commit 39874b7 (localhost撲滅) が commit 1287234 で上書きされた
- 手動で `git reset --hard` と `--force push` による復旧が必要になった

**正しい方法:**
- GitLab UI から手動でパイプラインを実行
- または `git commit` + `git push` で自然にパイプラインをトリガー
