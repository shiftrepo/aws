üì¶ CA Certificate Installation

‚ö†Ô∏è To eliminate browser warnings, import the CA certificate (ca.crt) into your system or browser trust store.

Step 1: Get CA Certificate

Option A: Copy from server: /root/aws.git/container/claudecode/CA/certs/ca/ca.crt

Option B: Export from this browser:

  1. Chrome/Edge: Click warning icon ‚Üí "Certificate (Invalid)" ‚Üí Details tab ‚Üí Select "OnPremise-CA-Root" ‚Üí Export as "Base64-encoded X.509 (.CER)"
  2. Firefox: Click warning ‚Üí "View Certificate" ‚Üí Find "OnPremise-CA-Root" ‚Üí Download ‚Üí Choose "PEM (cert)"
  3. Save as ca.crt

Important: Export format should be .crt or .pem (Base64-encoded X.509), NOT .pfx or .p12

Step 2: Install Certificate

Linux / Ubuntu

sudo cp ca.crt /usr/local/share/ca-certificates/onpremise-ca.crt
sudo update-ca-certificates

Windows

certutil -addstore -f "ROOT" ca.crt

Or double-click ca.crt and follow the Certificate Import Wizard.

macOS

sudo security add-trusted-cert -d -r trustRoot \
    -k /Library/Keychains/System.keychain ca.crt

Browser (Firefox)

  1. Open Firefox Settings
  2. Go to "Privacy & Security"
  3. Scroll to "Certificates" ‚Üí Click "View Certificates"
  4. Go to "Authorities" tab ‚Üí Click "Import"
  5. Select ca.crt file
  6. Check "Trust this CA to identify websites"

Browser (Chrome / Edge)

Chrome and Edge use the system certificate store. Install the CA certificate using the OS-specific method above.
